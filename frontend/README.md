# CampusPro Frontend

This React + Tailwind frontend connects to the Django CampusPro backend at `http://127.0.0.1:8000/api/`.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Features

- JWT login with role-based dashboard
- Dark theme with sidebar navigation
- Student records, admission dashboard, attendance, fees, exams, HRMS, payroll, and library pages
- API-driven tables with search and sort
- Toast notifications for async actions

## Notes

If `npm` is not installed locally, install Node.js first and then rerun the commands above.
